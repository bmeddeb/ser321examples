FunWebServer End Points:

Title Endpoint
Random Picture /random
This endpoint responds with a random number between 1 and 10 when a GET request is made.
JSON Greetings Provider /json
This endpoint responds with a JSON object that contains the message "Hello World" when a GET request is made.
Integer Multiplier /multiply?num1=5&num2=10
This endpoint allows a user to multiply two numbers. The user provides the numbers as query parameters num1 and num2 in the URL, and the server responds with the product of these two numbers.
GitHub Lister /github?query=users/username/repos
This endpoint fetches a list of repositories for a specified user from the GitHub API. The user replaces username with the desired GitHub user's username. The server then makes a request to the GitHub API and prints out the name and URL of each repository.
GitHub Events Reporter /githubActivity?user=<username>
This endpoint fetches a list of public events for a specified user from the GitHub API. The user replaces username with the desired GitHub user's username. The server then makes a request to the GitHub API and prints out the type of each event along with the corresponding repository name and URL. This can be used to see the recent activity of a GitHub user.
Password Generator /pass?l=&OPT=<1 2 or 3>
The "/pass" endpoint accepts two query parameters: l (length of password), and OPT (exclude certain characters). It generates and returns a password based on these parameters.: 1: Exclude uppercase letters. 2: Exclude special characters. 3: Exclude numbers. Users can also combine the options to exclude multiple character types. For example, the input "31" would exclude both numbers and uppercase letters from the password. This class is responsible for processing the HTTP request, validating the query parameters, generating the password according to the user's specifications, and returning the password as the HTTP response. It employs randomness and a pre-set character set to generate the password.

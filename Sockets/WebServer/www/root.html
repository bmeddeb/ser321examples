You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=VALUE1&num2=VALUE2 -- multiplies the value of num1 and num2. Replace VALUE1 and VALUE2 with the numbers you want to multiply
  • /github?query=users/USERNAME/repos -- returns Github profile data for the specified user. Replace USERNAME with the Github username.
    Example: /github?query=users/amehlhase316/repos
  • /githubActivity?user=USERNAME -- returns the Github activity for the specified user. Replace USERNAME with the Github username.
    Example: /githubActivity?user=amehlhase316

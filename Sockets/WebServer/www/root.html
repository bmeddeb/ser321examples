FunWebServer End Points:

Title Endpoint
Random Picture /random
This endpoint responds with a random Picture.
JSON Greetings Provider /json
This endpoint responds with a JSON object
Integer Multiplier /multiply?num1=5&num2=10
This endpoint allows a user to multiply two numbers.
GitHub Lister /github?query=users/username/repos
This endpoint fetches a list of repositories for a specified user from the GitHub AP.
GitHub Events Reporter /githubActivity?user=<username>
This endpoint fetches a list of public events for a specified user from the GitHub API. .
Password Generator /pass?l=&OPT=<1 2 or 3>
This endpoint generates a custom password. Use parameters 'l' for length and 'OPT' for character exclusions (1: Uppercase, 2: Special characters, 3: Numbers) To exclude upper case choose 1, you can combine exclusions example 21 will exclude Special characters and Upper case etc... Will default to 6 characters without options or when supplied wrong parameters
